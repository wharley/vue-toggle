# vue-toggle
 A simple toggle component using `Vue-observable` from Vuejs

 ![Screenshot of vue-toggle](/src/assets/docs/vue-toggle.gif)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```
